Displaying Pixel Fonts

Unlike normal contoured fonts, pixel fonts require special treatment to get them to display correctly.

Normal fonts can be displayed at any size, but pixel fonts must be rendered at a multiple of size 16 to look correct:

You also have to make sure antialiasing is disabled, or the font will look blurry like this:

Letter Kerning

Every font has a list of kerning pairs, which are numbers associated with certain pairs of characters. These tables are built into the provided TTF files, and if you are using the other assets, they are provided in the fonts' associated data files.

This kerning number represents how much to x-offset the right character when it is positioned immediately after the left. When using custom renderers, it is recommended (when using a non-monospace font) to offset your cursor by these values to make words fit together more tightly, improving readability.
